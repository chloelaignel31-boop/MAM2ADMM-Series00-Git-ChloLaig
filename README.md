# Introduction

## Objectif du dépôt

- Pratiquer les commandes de base de Git : `clone`, `add`, `commit`, `push`, `pull`.  
- Expérimenter avec les branches et les merges.  
- Découvrir les bonnes pratiques de gestion de dépôt sur GitHub.
